# 蓮華堂カウンセリング - 七観音の智慧 UI/UX設計

## アプリコンセプト
ユーザーの悩みを「七難」に分類し、それぞれに対応する観音様とのAI対話を通じてカウンセリングを行うモバイルアプリ。観音経の智慧を現代のメンタルケアに活かす。

## 画面構成

### 1. ホーム画面（Welcome Screen）
- **目的**: アプリの世界観を伝え、ユーザーを穏やかに迎え入れる
- **コンテンツ**:
  - 蓮華堂のロゴ/アイコン
  - 「心の悩みを観音様にお話しください」というメッセージ
  - 「相談を始める」ボタン（大きく、タップしやすい）
  - 過去の相談履歴へのリンク（小さく控えめに）
- **デザイン方向性**: 
  - 背景は柔らかいグラデーション（白〜淡い青紫）
  - 仏教的な要素を取り入れつつ、モダンでクリーンなデザイン
  - フォントは読みやすい明朝体とゴシック体の組み合わせ

### 2. 悩み入力画面（Input Screen）
- **目的**: ユーザーが自由に悩みを入力できる安心感のある空間
- **コンテンツ**:
  - 「どのようなことでお悩みですか？」という優しい問いかけ
  - 複数行のテキスト入力エリア（大きく、十分なスペース）
  - プレースホルダー: 「例: 仕事でのストレスが溜まっています...」
  - 「観音様に相談する」送信ボタン
  - 戻るボタン（左上）
- **デザイン方向性**:
  - 入力エリアは柔らかい枠線、背景は白
  - 文字数制限は表示しない（自由に書いてもらう）

### 3. 観音様選択画面（Kannon Selection Screen）
- **目的**: AIが分類した「難」と対応する観音様を提示
- **コンテンツ**:
  - 「あなたの悩みは【○○の難】と感じられます」というメッセージ
  - 対応する観音様の名前と簡単な説明
  - 観音様のイラスト/画像（静的または3Dモデル）
  - 「この観音様とお話しする」ボタン
  - 「別の観音様を選ぶ」リンク（任意で変更可能）
- **デザイン方向性**:
  - 観音様の画像を中心に配置
  - 説明文は短く、読みやすく
  - 色は観音様ごとに微妙に変える（火=赤系、水=青系など）

### 4. カウンセリング対話画面（Chat Screen）
- **目的**: 観音様とのAI対話を通じてカウンセリングを行う
- **コンテンツ**:
  - 上部: 観音様の名前とアイコン
  - 中央: チャット形式の対話履歴（ユーザーと観音様のメッセージ）
  - 下部: テキスト入力欄と送信ボタン
  - 戻るボタン（左上）で観音様選択画面に戻る
- **デザイン方向性**:
  - ユーザーのメッセージ: 右寄せ、淡い色の吹き出し
  - 観音様のメッセージ: 左寄せ、観音様のカラーの吹き出し
  - スクロール可能なメッセージリスト
  - 入力欄は固定、キーボード表示時も見やすく

### 5. 相談履歴画面（History Screen）
- **目的**: 過去の相談を振り返る
- **コンテンツ**:
  - 過去の相談のリスト（日付、対応した観音様、悩みの要約）
  - タップで詳細（過去の対話）を表示
  - 削除機能（スワイプまたは長押し）
- **デザイン方向性**:
  - リスト形式、カード型デザイン
  - 観音様のアイコンを小さく表示

## 七難と観音様の対応

| 難の種類 | 観音様 | 色テーマ | 特徴 |
|---------|--------|---------|------|
| 火の難 | 聖観音 | 赤・オレンジ | 怒りや情熱の問題 |
| 水の難 | 千手観音 | 青・水色 | 感情の溢れ、悲しみ |
| 風の難 | 馬頭観音 | 緑・黄緑 | 不安定、迷い |
| 鬼の難 | 十一面観音 | 紫・藤色 | 恐怖、トラウマ |
| 刀の難 | 如意輪観音 | 金・黄色 | 対人関係の衝突 |
| 鎖の難 | 不空羂索観音 | 灰・銀色 | 束縛、自由の欠如 |
| 怨の難 | 准胝観音 | 深紅・茶色 | 恨み、過去への執着 |

## ユーザーフロー

1. **アプリ起動** → ホーム画面
2. **「相談を始める」タップ** → 悩み入力画面
3. **悩みを入力して送信** → AI分析中（ローディング表示）
4. **分析完了** → 観音様選択画面（推奨される観音様を表示）
5. **「この観音様とお話しする」タップ** → カウンセリング対話画面
6. **対話を続ける** → メッセージを送受信
7. **対話終了** → ホーム画面に戻る（履歴に保存）

## カラーパレット

- **プライマリカラー**: 深い青紫（#5B4A8F）- 観音様の神聖さを表現
- **セカンダリカラー**: 淡いピンク（#F4E4E8）- 慈悲と優しさ
- **背景**: 白〜淡いグレー（#FAFAFA）
- **テキスト**: ダークグレー（#333333）
- **アクセント**: 金色（#D4AF37）- 仏教的な荘厳さ

## フォント

- **見出し**: 明朝体（優雅さ）
- **本文**: ゴシック体（読みやすさ）
- **ボタン**: ゴシック体（明瞭さ）

## インタラクション

- **ボタンタップ**: 軽いスケールダウン（0.97）+ 触覚フィードバック
- **画面遷移**: フェードイン/アウト（250ms）
- **メッセージ送信**: スムーズなスクロールアニメーション
- **ローディング**: 蓮の花が開くアニメーション（イメージ）

## 技術的考慮事項

- **AI分類**: ユーザーの入力テキストをサーバー側のLLMで分析し、七難のいずれかに分類
- **対話生成**: 観音様ごとに異なるペルソナ（システムプロンプト）を設定し、カウンセリング対話を生成
- **データ保存**: AsyncStorageに相談履歴を保存（ローカルのみ、クラウド同期なし）
- **プライバシー**: ユーザーの悩みは端末内に保存、外部には送信しない（AI処理時のみサーバー経由）
